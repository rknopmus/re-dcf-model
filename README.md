# RE DCF Model

Web app en React + Vite para cargar un Excel `RE_DCF.xlsx`, leer la pestaña `Inputs`, calcular un modelo DCF inmobiliario con iteración circular y mostrar gráficos de Income, Expenses, NOI y DCF.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Deploy

Funciona directamente en Vercel:

- Build command: `npm run build`
- Output directory: `dist`
